# Architecture Concepts

## MemU Framework
- MemU is an agentic memory framework specifically designed for AI agents, enhancing memory capabilities beyond traditional retrieval-augmented systems.
- It organizes, categorizes, and maintains memories semantically.

## Memory Organization
- MemU features layered memory organization structured into:
  - **Memory Items**
  - **Categories**
  - **Types**: Profile, Event, Knowledge, Behavior.
